Miku supports GNOME 3.18 and above and it is far from ready. Do not fork it yet!

##Building

Install dependencies

`
npm install -g gulp
`

Clone it.

`
git clone https://github.com/Miku-GNOME/twoFacedLovers-shell.git
`

Install modules

`
cd twoFacedLovers-shell
`

`
npm install
`

Run sass watcher

`
gulp sass:watch
`
