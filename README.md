##Building

Install dependencies

`
npm install -g gulp
`

Clone it.

`
git clone https://github.com/Miku-GNOME/twoFacedLovers-shell.git
`

Install modules

`
cd twoFacedLovers-shell
`

`
npm install
`

Run SASS watcher

`
gulp sass:watch
`
